### Some Important Drectories

| Container Location                   | Local Location        | Usage                                     |
|--------------------------------------|-----------------------|-------------------------------------------|
| `/var/opt/gitlab`                    | $GITLAB_HOME/data     | For storing application data              |
| `/var/log/gitlab`                    | $GITLAB_HOME/logs     | For storing logs                          |
| `/etc/gitlab`                        | $GITLAB_HOME/config   | For storing gitlab config files           |
| `/var/opt/gitlab/backup`             | $GITLAB_HOME/backups  | For storing backups                       |
| `/etc/gitlab/initial_root_password`  |                       | Initial root pass (better to not config)  |
